# Agentic AI Developer Final Project

This project demonstrates a simple agentic AI system that interacts with a simulated environment.

## Files
- `main.py` : Entry point to run the agent.
- `agent.py` : Agent decision-making and learning logic.
- `environment.py` : Simulated environment with states, actions, and rewards.
- `utils.py` : Helper functions.
- `requirements.txt` : Dependencies.

## How to Run
```bash
pip install -r requirements.txt
python main.py
